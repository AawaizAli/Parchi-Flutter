# Parchi - Student App

A Flutter-based mobile application for students to discover and redeem discounts from verified merchants using a secure token-based system.

## ðŸš€ Features

- **Student Verification**: Secure KYC process with document upload
- **Unique Parchi ID**: Each student gets a unique identifier (QR + alphanumeric)
- **Discount Discovery**: Browse available offers from partner merchants
- **Flexible Redemption**: 
  - Walk-in: Scan QR code at restaurant
  - Dine-in: Generate discount codes or scan table QR
- **Loyalty System**: 
  - Top Merchants leaderboard
  - Bonus discounts after 5 redemptions at same merchant
  - Founders Club referral program
- **Redemption History**: Track all past transactions
- **Personal Analytics**: View total savings and redemption stats

## ðŸ”§ Installation

### Prerequisites
- Flutter SDK 3.xx or later
- Android Studio / VS Code
- iOS: macOS with Xcode (for iOS builds)

### Setup Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/parchi-flutter-app.git
   cd parchi-flutter-app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure environment**
   - Copy `.env.example` to `.env`
   - Update with your configuration:
     ```
     BASE_URL=your_backend_api_url
     SUPABASE_URL=your_supabase_url
     SUPABASE_ANON_KEY=your_supabase_anon_key
     ```

4. **Run the app**
   ```bash
   flutter run
   ```
